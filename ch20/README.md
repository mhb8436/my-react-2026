## create directoy
```bash
src
    components  # UI컴포넌트를 모아놓는 디렉토리
    data        # 쇼핑몰에서 판매되는 제품정보를 모아놓은 디렉토리
    lib         # UI에서 관련된 라이브러리 파일을 모아놓는 디렉토리
    types       # 타입만 모아서 보관하는 디렉토리 
    App.tsx     # 우리 프로그램의 엔트리 포인트(문)

```
